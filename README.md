# slider
Arduino Slider Control 

Pins used:  <br />
StepPin : GPIO 9  <br />
DirPIn  : GPIO 8   <br />
Joystick : Analog 14,15  <br />
LCD : GPIO 2, 3, 4, 5, 6, 7  <br />
